Font Size

SVGに出力されるテキストのフォントサイズをピクセル単位で指定します。デフォルト値は14です。

設定可能範囲: 8pxから72px

Base Background Color

SVG全体の背景色を指定します。カラーピッカーで色を選択できます。

「Transparent」チェックボックスをオンにすると、背景が透明になります。この設定は、他のアプリケーションでSVGを使用する際に背景が不要な場合に便利です。

デフォルト値: #ffffff (白)

Node Background Color

PADの各ノード（ブロック）の背景色を指定します。カラーピッカーで色を選択できます。

「Transparent」チェックボックスをオンにすると、ノードの背景が透明になります。これにより、Base Background Colorが透けて表示されます。

デフォルト値: #ffffff (白)

Text Color

SVGに出力されるテキストの色を指定します。カラーピッカーで色を選択できます。

デフォルト値: #000000 (黒)

List Render Type

リスト構造（複数のノードが連続するブロック）の描画方法を選択します。

  • TerminalOffset: リスト内の先頭、末尾ある端子ノードが少しオフセットされて描画されます。
  • Original: リスト内の各ノードが、一直線に描画されます。

デフォルト値: TerminalOffset
